# 音频转文字工作流 · Audio Transcript Pipeline

把一整个文件夹里的长音频，批量转成可阅读的 Markdown 文字稿。

支持 **长音频切分、国内 ASR 转录、断点续跑、集中导出、可选清理和阅读笔记**。适合课程音频、直播回放、会议录音、访谈录音等批量处理场景。
